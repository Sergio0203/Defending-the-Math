**Defending the Math** é um jogo de defesa e raciocínio lógico desenvolvido nativamente para iOS. O objetivo é proteger sua base de hordas de inimigos resolvendo cálculos matemáticos em tempo real. Cada acerto é um ataque; cada erro é uma brecha na sua defesa.

---

## Mecânicas do Projeto

O jogo foi construído utilizando o motor **SpriteKit**, focando em performance e resposta tátil:

- **Game Loop:** Gerenciamento de estados de jogo (Menu, Playing, GameOver) via `SKScene`.
- **Sistemas de Spawn:** Inimigos com dificuldades dinâmicas baseadas em operações aritméticas.
- **Feedback Visual:** Uso de partículas e transições de cena para melhorar a experiência do usuário (UX).
- **Lógica de Colisão:** Implementação de `SKPhysicsContactDelegate` para detecção de dano na base.

# Detalhes Técnicos

Como um projeto focado em **Clean Code** e **Performance**, o código segue padrões de engenharia de software:

* **Arquitetura:** Separação entre a lógica de cena (`GameScene`) e os modelos de dados.
* **Gestão de Assets:** Carregamento otimizado para garantir 60 FPS estáveis.
* **Escalabilidade:** Sistema desenhado para permitir a adição fácil de novos operadores (multiplicação, divisão).

## Como Executar

1. Clone o repositório:
   ```bash
   git clone [https://github.com/Sergio0203/Defending-the-Math.git](https://github.com/Sergio0203/Defending-the-Math.git)
   ```
2. Abra o arquivo `.xcodeproj` no **Xcode**.
3. Selecione o simulador de iPhone (iOS 16.0+).
4. Pressione `Cmd + R` para buildar.
